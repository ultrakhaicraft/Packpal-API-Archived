# Packpal-API-Archived

## Introduction
An user-oriented storage service API System, where the user can book a request to storage their items to other people storage or can create their own storage management for other to use it. It also support real-time update through SignalR

Current Project Status: Archived. This git repository will be serve as an archived of what our team manage to accomplished

## Technology 
-Language: C# (.NET API + SignalR)
-Third party: PayOS, Cloudinary, Firebase
-Deployment: Docker
-Database: EFCore, MySQL
